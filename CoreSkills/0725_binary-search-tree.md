# 004 - Design Binary Search Tree

**Date:** 2025-07-25
**Platform:** Core Skills https://neetcode.io/problems/binarySearchTree
**Difficulty:** Easy

## Problem Description

Binary Tree を実装する

## Key Learnings

- Binary Tree: 辞書の一種。
  - 親子関係がある
    - 「左の子 < 親 < 右の子」と決められている
  - Root（＝トップノード）, Leaf Nodes
  - cycle を持たない, 同じ段の横はつながらない sibling nodes
  - Depth: root→children→children なら、Depth は 3
