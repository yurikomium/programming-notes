# Binary Search

**Date:** 2025-09-01
**Platform:** NeetCode 150 https://neetcode.io/problems/binary-tree-diameter?list=neetcode150
**Difficulty:** Easy

## Problem Description

Diameter = 木の中の任意の 2 つのノード間の最長パスを求める

## Key Learnings

- 最長パスと深さ
  - 最長パス：ルートを通らなくてもいい
  - 深さ：必ずルートを起点とする

## Memo

- カプセル化: 関数を入れ子構造にする
  - カプセル化することで、内側の変数やヘルパー関数はメソッド内のみで使用される
    - -> 外部からの干渉されない, コードの責任範囲が明確
