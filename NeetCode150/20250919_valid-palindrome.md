# Binary Search

**Date:** 2025-09-19
**Platform:** https://neetcode.io/problems/is-palindrome?list=neetcode150
**Difficulty:** Easy

## Problem Description

- Two Pointers を使って両側から内側に向かって比較することで、1 回の走査で判定できる

## Key Learnings

- Two Pointers を使うと、計算量が少なくて済む
- 不要な文字を while で先に飛ばす
- 文字判定は標準メソッドを使える
